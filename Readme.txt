Welcome to RogueLike!

You can start the game by simply starting "RogueLike.exe".
Explore the dungeon that is always newly generated for you!
Everything is turn based, so no hurry!

______________________________________________________________________________

Controls:

Movement + aiming: ----------------------------- Arrow keys
Wait a turn + target yourself: ----------------- Space bar
Cast spell: ------------------------------------ C
Use item: -------------------------------------- U
Throw item: ------------------------------------ T
Choose item/spell: ----------------------------- 1 - 9
Cancel: ---------------------------------------- Esc
